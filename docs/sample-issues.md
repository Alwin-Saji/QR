# Sample Issues

This document contains examples of common issues and how they should be formatted.

## Bug Report Example

**Title**: [BUG] Camera capture button disabled on iOS Safari
**Labels**: `bug`

**Describe the bug**
When tapping the camera button on an iPhone 13 running Safari, the button appears disabled and nothing happens.

**To Reproduce**
1. Open ARC event link in Safari on iOS
2. Tap the camera button at the bottom of the screen
3. See that nothing happens.

**Expected behavior**
The native camera interface should open.

**Environment**
 - OS: iOS 16
 - Browser: Safari

## Feature Request Example

**Title**: [FEATURE] Add ability to delete photos
**Labels**: `enhancement`

**Is your feature request related to a problem?**
Sometimes guests accidentally upload blurry photos and want to remove them.

**Describe the solution you'd like**
An authenticated user (event owner) or the original uploader should have a delete button on the photo overlay.
